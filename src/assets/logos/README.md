# Logo Assets

Put brand logo files here as PNG, JPG, JPEG, or WebP.

Recommended filenames:

- `canon.png`
- `nikon.png`
- `sony.png`
- `fujifilm.png`
- `leica.png`
- `panasonic.png`
- `olympus.png`
- `ricoh.png`
- `pentax.png`
- `hasselblad.png`
- `sigma.png`
- `tamron.png`
- `zeiss.png`
- `voigtlander.png`
- `laowa.png`
- `viltrox.png`

The app will fall back to a clean text wordmark when a logo image is missing.
